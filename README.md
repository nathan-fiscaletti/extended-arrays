# Extended Arrays

> ---
> > **Extended Arrays** is a simple library used to extend upon PHP's built in arrays.
> >
> > **Hint**: Extended Arrays is available through [Composer](https://getcomposer.org). `composer require nafisc/extended-arrays`.
